Introduction

 [toc]
M3U creator is a small little addon that allows you to create custom playlists that are supported by both Soundplay and CL-Amp. The playlists mime type are set to "text/x-playlist" and are formated with one input filename per line. It's pretty handy if you've just encoded a CD to [MP3/OGG] and you now want a playlist for it. Now you don't have to add each song to Soundplay's playlist editor and worry about sorting, etc. You can create it with the same user friendly interface that you used to encode the files in the first place!




Encoder Settings

 [toc]
M3U Creator does not provide a settings menu.




File Path and Name Pattern

 [toc]
When specifing the File Path and Name Pattern, you can specify one that has multiple files that have the same Save As attribute. What M3U Creator will do is for the first input file it runs across with this output file name, it will create a new file (if it doesn't already exist) or delete the contents of the file (if it already exists) and add the full path to the input file as the first entry in the output file. Each subsequent file that M3U Creator encodes to the same output file will be appended to the file. This process occurs every time you press the Encode button on FlipSide A.E.




Details Editor

 [toc]
All file details are ignored except for the input and output files.




The File List

 [toc]
Files are encoded in the order that they appear in the File List when you press the encode button. This means you can sort by Artist then year then album, then track number if you like and the m3u file will have this order. Pretty cool huh?
