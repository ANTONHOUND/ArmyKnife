Introduction

 [toc]
This add-on along with FlipSide A.E. gives you the ability to completely throw away that other GUI encoder that's showing it's age. You get all the benefits of FlipSide A.E.'s user friendly GUI with the encoding speed of gogo. This release of MP3 GoGo has been tested with gogo version 2.39, if you are using an older version, it is recomended that you upgrade. The binary is available on BeBits.




Encoder Settings

 [toc]
Here are a brief explination of the settings available to you. This covers all of the good stuff that gogo would allow you to do on the command line.
  • Bitrate - Allows you to choose from either a fixed bitrate, or use one of the levels of varible bitrate compression.
  • Output Format - Allows you to specify the output format of the file (stereo,mono,j-stereo).
  • Psycho Acoustics - Turns on and off Psycho Acoustic compression of the file.




File Path and Name Pattern

Details Editor

 [toc]
MP3 GoGo writes both BeOS attributes, and ID3 v1.1 tags to the newly encoded file. Remember, if you specified a genre other than an ID3 v1.1 genre, the ID3 tag will reflect the "Other" genre, but the BeOS attribute will be what you specified. Very compatible with The Army Knife if you wish to update things later!
